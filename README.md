# Personal Portfolio Website

A responsive personal portfolio website built using **HTML5, CSS3, and JavaScript** to showcase my skills, education, projects, internships, and contact information.

## Project Description

This portfolio website is designed to present my professional profile in a modern and responsive way. It follows HTML5 semantic structure and responsive web design principles to provide a seamless experience across desktop, tablet, and mobile devices.

## Features

- Responsive design (Mobile, Tablet, Desktop)
- Semantic HTML5 structure
- CSS Grid and Flexbox layouts
- Responsive navigation menu
- About Me section
- Education section
- Skills section
- Projects showcase
- Internship section
- Contact form with validation
- Form submission with Thank You page
- Smooth scrolling
- Hover effects and animations
- Back to Top button

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- CSS Variables
- CSS Grid
- Flexbox
- Font Awesome
- Git & GitHub

1. Download or clone the repository.

```

```

2. Open the project folder.

3. Open `index.html` using VS Code Live Server.

4. Explore the portfolio website.
